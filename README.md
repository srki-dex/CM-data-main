# CYRILLIC MATRIX

**(You need to know to run: Downloading Files, Zip / UnZip Files, Cut / Move and Copy / Pate Operations)**

<img src="logo.png" width="100%" />

Cyrillic Matrix is a NW.js web / js app that acts as a screen saver on all monitors

## Installation

1. Download [NW.js](https://nwjs.io/) NORMAL edition and Extract ZIP

2. Download ZIP [SS-data](https://github.com/srki-dex/CM-data/archive/refs/heads/main.zip)

3. Extract ZIP to "CM-data-main" folder (make if needed) inside NW.js main folder.

4. CUT / MOVE "package.json" from "CM-data-main" UP to the folder above. 

5. RUN NW.js app (.exe, .bin ...)
